# Frontend

## Environment Variables
Create a `.env` file in the frontend directory:

```env
VITE_API_URL=http://localhost:3000/api/v1
```

## Installation

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

Open http://localhost:3001

## Build

```bash
npm run build
```
